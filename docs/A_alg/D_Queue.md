# 单调队列 Monotonous Queue

- 使用单调队列，实现窗口内最大值或最小值的更新结构
- 能够方便获取某个区间内（滑动窗口）的最大值或最小值

# 单调栈 Monotonous Stack

- 使用单调栈，实现最临近的大于或小于关系的更新结构
- 能够方便获取大于或小于当前值的最邻近的位置
